# 🎬 Absolute Cinema — сайт бронирования билетов в кино

[![GitHub Pages](https://img.shields.io/badge/Deployed-GitHub%20Pages-blue)](https://asikoshhh.github.io/absolute-cinema/)
[![Jira](https://img.shields.io/badge/Project%20Management-Jira-blue)](https://projectcinema.atlassian.net)

## 📌 О проекте

**Absolute Cinema** — это веб-приложение для бронирования билетов в кинотеатр. Проект разработан в рамках курса **«Управление проектами информационных систем»**.

Сайт позволяет пользователю:
- Просматривать афишу фильмов
- Читать описание и смотреть постеры
- Выбирать сеанс (время, зал, цена)
- Бронировать места в интерактивном зале
- Подтверждать бронирование в корзине

---

|------|-----|-------------|

| **Разработчик** | [Асанали] | Логика бронирования, карта зала, корзина, localStorage |
| **Дизайнер** | [Байболсын] | Цветовая схема, логотип, макеты, иконки |

---

## 🛠 Технологии

- **Frontend**: HTML5, CSS3, JavaScript (ES6)
- **Хранение данных**: localStorage
- **Управление проектом**: Jira, Scrum
- **Деплой**: GitHub Pages

---

## 🎯 Функционал

- ✅ Афиша с 12 фильмами (постеры, жанр, возрастной рейтинг)
- ✅ Страница фильма (описание, мета-информация, расписание)
- ✅ Интерактивная карта зала (8 рядов × 10 мест)
- ✅ Выбор мест с визуальной обратной связью
- ✅ Корзина с подсчётом суммы
- ✅ Сохранение занятых мест (даже после перезагрузки)
- ✅ Адаптивный дизайн
- ✅ Футер с контактами

## 📋 Управление проектом (Jira)

Проект вёлся в **Jira** с использованием **Scrum**:

- **1** — проектирование, ТЗ, макеты
- **2** — вёрстка главной страницы и страницы фильма
- **3** — логика бронирования и корзина
- **4** — интеграция, тестирование, деплой

Все задачи распределены по ролям (`PM`, `DEV`, `DES`) с помощью компонентов Jira.
