# Техническое задание на разработку сайта бронирования билетов в кино

## 1. Цель проекта
Разработать веб-сайт для бронирования билетов в кинотеатр.

## 2. Участники команды
| Роль | Имя | Обязанности |
|------|-----|--------------|
| PM | [Асанали Кемелбаев] | Управление проектом, вёрстка страниц, интеграция |
| Разработчик | [Асанали Кемелбаев (Второй аккаунт) ] | Логика бронирования, карта зала, корзина |
| Дизайнер | [Байболсын Конысбай ] | Макеты, цветовая схема, иконки |

## 3. Функциональные требования
3.1. Афиша фильмов (постеры, название, кнопка "Выбрать билет")
3.2. Страница фильма (описание, расписание сеансов)
3.3. Карта зала (интерактивный выбор мест)
3.4. Корзина с выбранными билетами
3.5. Страница подтверждения бронирования

## 4. Технологии
- HTML/CSS/JavaScript
- Хранение данных: localStorage
- Репозиторий: GitHub

## 5. План по спринтам
| Спринт | Задачи | Длительность |
|--------|--------|--------------|
| Спринт 1 | Проектирование, создание макетов | 1 неделя |
| Спринт 2 | Вёрстка главной страницы и страницы фильма | 1 неделя |
| Спринт 3 | Логика бронирования и корзина | 1 неделя |
| Спринт 4 | Тестирование и презентация | 3 дня |

## 6. Критерии успеха
- Сайт открывается в браузере
- Можно выбрать фильм, сеанс, место
- Билет появляется в корзине
- После подтверждения места становятся занятыми
